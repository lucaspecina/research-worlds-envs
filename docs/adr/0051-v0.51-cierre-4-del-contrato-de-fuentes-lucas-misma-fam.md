# ADR 0051 — CIERRE #4 del contrato de fuentes (Lucas; misma familia que el sesgo, un momento más arriba): identificabilidad de σ_med — DECISIÓN (i) mediciones REPLICADAS

- **Versión**: v0.51
- **Fecha**: 2026-07-04
- **Estado**: Vigente

---

- **El problema**: el ruido zero-mean hace recuperable la MEDIA, pero la descomposición de VARIANZA no es identificable con una medición por unidad (`Var_obs = σ_true² + σ_med²`: una ecuación, dos incógnitas). Consecuencias sin cierre: **techo R=1 inalcanzable para cualquier agente** (mini-versión del problema de anclas de v2) + penalización de dispersión en TODOS los ítems, también para el ingenuo (compresión de escala).
- **DECISIÓN: (i) fuente de mediciones REPLICADAS** — dos lecturas por unidad, n chico, costo declarado → `σ_med` identificable vía `Var(Y₁−Y₂)/2`. *Testea una skill real (estimar el error del propio instrumento) y mantiene el techo alcanzable.* La alternativa (ii) (declarar σ en el brief — spec parcial) queda registrada como más barata y menos skill; no elegida.
- **Gate del certificado**: la recuperabilidad debe mostrar **techo ≈ world.py bajo la elección** (si el techo no se recupera con la fuente replicada en mano, el cierre falló y se investiga antes de seguir).
- **Con esto: CERO decisiones entre el mundo 3 y el código. El reloj de ~2 sesiones CORRE.**
