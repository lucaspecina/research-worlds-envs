- **Versión**: v0.49
- **Fecha**: 2026-07-04
- **Estado**: Vigente

---

- **(1) LINT → PRE-COMMIT (hecho)**: hook versionado en `hooks/pre-commit` (mismos tests de `test_doc_headers.py`, un paso antes) + `git config core.hooksPath hooks` (documentado en CLAUDE.md como paso post-clone). El CI queda como segunda red. *Cuatro capturas post-commit en un día = tasa base medida; la misma doctrina — estructura, no memoria — aplicada más temprano en el pipeline.*
- **(2) PLAN CORTO — mundo 3 (`selection_bias_v0`, working name; suite muestreo/canal):**
  - **Mecanismo (limpio)**: SCM chico — `driver` (settable 0–10) → `outcome`; `signal` correlacionada con `outcome` solo vía `driver` (sin arista directa). Columnas: `driver, signal, outcome` (+1 señuelo). El mecanismo NO instala la correlación trampa.
  - **Operadores (por PRIMERA vez en capa fuente, la anatomía §1 al fin implementada)**: `collider_seleccion` (muestreo): la fuente barata solo contiene filas que pasaron un filtro `f(signal, outcome) > umbral` (knobs: peso de cada uno, dureza) → correlación espuria signal↔outcome INTRA-registro. `error_de_medicion` (canal): `outcome` se observa con ruido/sesgo declarado (knobs: σ_med, sesgo) — **y por la regla v0.9, TAMBIÉN en los experimentos** (el canal no se esquiva; test pre-registrado).
  - **Maquinaria nueva requerida (declarada de antemano, NO es dummy-ismo — es la §1 pendiente)**: `SourceSpec` gana su pipeline de corrupción declarado (filtro de selección + canal de medición) aplicado por el server en `observe()`/`experiment()` SOBRE `world.sample()` (que sigue limpio para el scorer). Contrato + validators + tests de wiring.
  - **El test del gemelo (pre-registro v0.48 vigente)**: el gemelo del collider ("la correlación seleccionada es real") debe salir de `build_standard_rivals` SIN código nuevo. **Predicción honesta**: va a FALLAR — `rival_twin` solo itera operadores `layer=="mechanism"`; los operadores de fuente no tienen ablación de params sino ablación de FUENTE (regenerar el pool sin el filtro). Si falla así, es el **dummy-ismo #7 (mechanism-only twins)**, se registra, y el fix es la generalización de la ablación por capa (fuente = re-muestrear pool con el operador off) — dirección fixes-fortalecen.
  - **Certificados**: mecanística vs (d-obs) sobre el pool corrompido; visibilidad del collider por **sub-batería observacional** (día uno, ruido local); brecha de teoría esperada ~0 (no hay latente — se registra, control de carga diferencial: este mundo carga en canal-sesgado + sondas-caras, ≥2 coordenadas ✓).
  - **Funcional**: el brief declarará una decisión con cola (umbral sobre `outcome`) → ambas monedas desde el día uno.
  - **Orden**: contrato fuentes → world.py + meta → brief (writer no-ciego = excepción bootstrap vigente) → derivación completa (test estructural) → L1 derivado → pre-registro de headroom → E0-probe.
