# ADR 0131 — Micro-batch N=5: yield mecánico 5/5, colapso de dominio y parámetros MEDIDO por el panel; maquinaria del fork verificada

- **Versión**: v1.31
- **Fecha**: 2026-07-10 (ventana autónoma; protocolo sellado pre-corrida en el commit del batch)
- **Estado**: Vigente

---

**Resultado del batch (ADR 0128 paso 2):** 4 encargos frescos (2 estrato B negativo, 2 estrato C
positivo-débil) + el piloto = **5/5 certifican ALL-PASS** con el verificador congelado, cero
reparametrizaciones, cero manos humanas. Primer indicio de yield: excelente — pero el panel sellado
detectó **exactamente los dos colapsos que r9 predijo**:

- **Colapso de DOMINIO**: 4/5 mundos son fábricas de papel (el generador stateless tiene un atractor
  fuerte hacia "secado de papel" para esta plantilla; el único distinto: horno de esmaltado cerámico).
- **Colapso de PARÁMETROS**: dentro de cada estrato el LLM eligió valores IDÉNTICOS (−2.4 todos los
  B; +1.4 todos los C — el número redondo del centro de la banda). Diversidad paramétrica espontánea:
  cero.
- **Panel conductual**: Pearson ≈ 0.999-1.0 en todos los pares; dos mundos con maxdiff **0.0000**
  (el mismo juego con otros nombres). La única separación conductual real (maxdiff ~0.13) la producen
  NUESTROS estratos (B vs C) — la diversidad vino del diseño, no del generador.

**Lectura sellada aplicada (r9, textual): "plantilla repetible, no diversidad".** Implicación de
diseño para la fábrica (a decidir con Lucas, alineada con la doctrina de dos niveles): los PARÁMETROS
debe muestrearlos el CÓDIGO (RNG seedeado dentro del estrato) — el LLM no aporta nada eligiéndolos y
colapsa al centro; el LLM queda para dominio+prosa, con la diversidad de dominio también DIRIGIDA
(pools o exclusión de dominios ya usados). La diversidad de nivel 1 (estructuras) sigue siendo
trabajo intelectual nuestro, como estaba escrito.

**Fork (maquinaria, smoke):** replay determinístico de una investigación donante VERIFICADO (9 turnos,
0 discrepancias, mismo presupuesto, cero costo LLM en el replay) + entrega bajo sufijo variante
funcionando. Anécdota n=1 (sin lectura): el mismo paquete de evidencia produjo entregas con R 0.13
(donante) / 0.04 (control) / 0.00 con sin-recorte −1.31 (pista) — la lotería de la entrega, visible.
El experimento fork REAL se pre-registra aparte con el banco de variantes de 0127.
